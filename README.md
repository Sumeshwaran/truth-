# 📰 Truth-Pro: AI Fake News Detector

Truth-Pro is an **AI-powered Fake News Detection platform** that analyzes news headlines or articles and predicts whether the information is **Real or Fake** using Artificial Intelligence and Natural Language Processing (NLP).

This project is designed to **combat misinformation and help users verify the credibility of online news content**. It can be used in social media platforms, journalism, education, and fact-checking systems.

---

## 🚀 Features

* 🔍 AI-powered news classification
* 📰 Detects **Fake vs Real news articles**
* ⚡ Instant prediction results
* 📊 Displays prediction **confidence score**
* 💻 Simple and responsive **web interface**
* 🧠 Uses **Natural Language Processing (NLP)**
* 🛡 Helps reduce **misinformation and fake news**

## 🧠 How It Works

1. The user enters a **news headline or article** in the input field.
2. The system processes the text using **Natural Language Processing techniques**.
3. The trained AI model analyzes linguistic patterns and features.
4. The system predicts whether the news is:

   * ✅ Real News
   * ❌ Fake News
5. The result and **confidence percentage** are displayed instantly.
